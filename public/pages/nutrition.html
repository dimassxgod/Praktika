Правильне харчування для фітнесу

Рекомендації щодо харчування, калькулятор калорій та плани харчування для досягнення ваших цілей

Харчування та тренування

Правильне харчування відіграє ключову роль у досягненні фітнес-цілей. Незалежно від того, чи хочете ви наростити м'язову масу, схуднути чи покращити загальний стан здоров'я, харчування становить близько 70% успіху.

Зниження ваги

Рекомендації щодо харчування для здорового зниження ваги без втрати м'язової маси. Створення дефіциту калорій із правильним балансом поживних речовин.

Набір м'язової маси

Стратегії харчування для ефективного набору м'язової маси, включаючи профіцит калорій та оптимальне споживання білка.

Здорове харчування

Загальні рекомендації щодо здорового харчування для підтримки енергії, покращення відновлення та загального самопочуття.

Макронутрієнти

Білки

Будівельний матеріал для м'язів Рекомендована норма: 1.6-2.2 г на кг ваги для тих, хто активно тренується.

Джерела: м'ясо, риба, яйця, молочні продукти, бобові, тофу.

Вуглеводи

Основне джерело енергії. Рекомендована норма: 3-7 г на кг ваги, залежно від інтенсивності тренувань.

Джерела: крупи, макарони, хліб, фрукти, овочі, бобові.

Жири

Важливими є для гормонального балансу. Рекомендована норма: 0,8-1,2 г на кг ваги.

Джерела: рослинні олії, горіхи, авокадо, жирна риба, оливки.

Калькулятор калорій

Використовуйте калькулятор для розрахунку приблизної кількості калорій та макронутрієнтів для ваших цілей.

Результати розрахунку:

БМР (Базовий метаболізм): 0 калорій на день
Денна норма калорій: 0 калорій на день
Рекомендоване споживання для вашої мети: 0 калорій на день
Білки: 0 г на день
Вуглеводи: 0 г на день
Жири: 0 г на день

Примітка: Результати є приблизними і можуть змінюватись в залежності від індивідуальних особливостей. Для складання персонального плану харчування радимо проконсультуватися з нашими фахівцями.

Поради щодо харчування

Час прийому їжі

Розподіляйте їжу на 3-5 разів на день. Харчуйте кожні 3-4 години для підтримки рівня енергії та метаболізму.

Водний режим

Пийте щонайменше 30 мл води на кг ваги. Збільшуйте кількість у спекотну погоду та при інтенсивних тренуваннях.

Підготовка їжі

Плануйте та готуйте їжу заздалегідь на кілька днів. Це допоможе дотримуватися плану харчування та уникнути нездорових перекусів.

Зразкові плани харчування

Ці плани є зразковими і можуть бути адаптовані під ваші особисті уподобання та потреби

План для схуднення

  • Сніданок: Омлет з 2 яєць з овочами, 1 шматок цільнозернового хліба
  • Перекус: Грецький йогурт з ягодами
  • Обід: Куряча грудка гриль (120г) з овочевим салатом та кіноа (50г)
  • Перекус: Яблуко та 15г мигдалю
  • Вечеря: Запечена риба (120г) з овочами на пару

План для набору м'язової маси

  • Сніданок: Вівсянка (80г) на молоці з бананом та 2 яйця
  • Перекус: Протеїновий коктейль з бананом та арахісовим маслом
  • Обід: Куряча грудка (150г) з рисом (100г) та овочами
  • Перекус: Сир (150г) з медом та горіхами
  • Вечеря: Яловичина (150г) з картоплею (150г) та овочевим салатом
  • Перед сном: Казеїновий протеїн або нежирний сир

План для підтримки форми

  • Сніданок: Мюслі (60г) з йогуртом та фруктами
  • Перекус: Фрукт та жменька горіхів
  • Обід: Індичка (120г) з гречкою (70г) та овочами
  • Перекус: Сирна запіканка або протеїновий батончик
  • Вечеря: Тушкована риба (120г) з овочами та бурим рисом (50г)
